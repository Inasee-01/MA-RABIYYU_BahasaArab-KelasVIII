# MA-RABIYYU_BahasaArab-KelasVIII
Madrasah 'Arabiyyah Online (MA'RABIYYU) merupakan sarana pembelajaran Bahasa Arab melalui website untuk siswa kelas VIII SMP Muhammadiyah Kota Semarang. MA'RABIYYU dikembangkan oleh salah satu mahasiswi Universitas Negeri Semarang yaitu Inas Rahimah dan Dosen Pembimbing bapak Hasan Busri, S.Pd.I., M.S.I.
